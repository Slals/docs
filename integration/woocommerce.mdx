## Intégration de Stan sur Woocommerce

Pour commencer vous devez avoir un compte Stan, rien de plus simple, rendez vous sur https://compte.stan-app.fr et entrez uniquement votre adresse email.

## Installer et configurer les deux extensions Stan

La solution Stan est composée de deux extensions faciles à installer !

<Note>
Si vous ne voyez pas de quoi il s'agit, transmettez cet article à votre développeur ou intégrateur web !
</Note>

[Stan Payment Gateway](https://wordpress.org/plugins/wc-stan-payment-gateway/) : extension de paiement sans carte, elle s’intègre parfaitement sur votre site Woocommerce.

[Stan Easy Connect](https://wordpress.org/plugins/stan-easy-connect/) : extension d’inscription et de connexion sans formulaire.

<Warning>
La connexion avec Stan fonctionne uniquement pour les Stanners. Ce sont les clients ayant l'application Stan installée sur leur téléphone !
</Warning>

<Steps>
  <Step title="Installer les extensions">
    Installez et activez [Stan Payment Gateway](https://wordpress.org/plugins/wc-stan-payment-gateway/) et [Stan Easy Connect](https://wordpress.org/plugins/stan-easy-connect/) sur votre site
  </Step>
  <Step title="Configurer Stan Pay">
    Rendez-vous sur la page « WooCommerce > Paiements > Gérer Paiements avec Stan »
  </Step>
  <Step title="Récupérer les clés API Stan Pay">
    Renseignez vos identifiants Stan Payment. Trouvez-les dans la section « Accès API » en vous connectant sur votre [Dashboard Stan](https://compte.stan-app.fr/integration/apikeys)
  </Step>
  <Step title="Configurer Stan Connect">
    Rendez-vous sur la page « Configurer » de Stan Easy Connect
  </Step>
  <Step title="Récupérer les clés API Stan Connect">
    Renseignez vos identifiants Stan Connect. Trouvez-les dans la section « Accès API » en vous connectant sur votre [Dashboard Stan](https://compte.stan-app.fr/integration/apikeys)
  </Step>
</Steps>

## Tester le paiement

Testez le paiement en cochant la case "Activer le mode test" sur la configuration de votre "Stan Payment Gateway" installée sur votre site. Sauvegardez et effectuez un paiement sur votre site, cela va réaliser un faux paiement.

<Warning>
    Pour avoir une installation fonctionnelle avec de vrais paiements, votre compte doit être activé par l'équipe Stan ! Nous faisons le nécessaire pour l'activer au plus vite. En attendant, vous pouvez désactiver l'extension "Stan Payment Gateway" depuis l'interface de méthodes de paiement Woocommerce.
</Warning>
