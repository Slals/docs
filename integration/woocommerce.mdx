## Intégration de Stan sur Woocommerce

Pour commencer vous devez avoir un compte Stan, rien de plus simple, rendez vous sur https://compte.stan-app.fr et entrez uniquement votre adresse email.

## Installer Stan Xpress Checkout

<Note>
Si vous ne voyez pas de quoi il s'agit, transmettez cet article à votre développeur ou intégrateur web !
</Note>

<a href="/downloads/wc-stan-xpress-checkout.zip" download>Télécharger Stan Xpress Checkout</a>

<Steps>
  <Step title="Installer les extensions">
    Installez et activez <a href="/downloads/wc-stan-xpress-checkout.zip" download>Stan Xpress Checkout</a> sur votre site Wordpress / Woocommerce
  </Step>
  <Step title="Configurer Stan Xpress Checkout">
    Rendez-vous sur la page « WooCommerce > Paiements > Gérer Paiements avec Stan »
  </Step>
  <Step title="Récupérer les clés API Stan Pay">
    Renseignez vos identifiants Stan Xpress Checkout. Trouvez-les dans la section « Accès API » en vous connectant sur votre [Dashboard Stan](https://compte.stan-app.fr/integration/apikeys)
  </Step>
</Steps>

## Tester le paiement

Testez le paiement en cochant la case "Activer le mode test" sur la configuration de votre "Stan Xpress Checkout" installée sur votre site. Sauvegardez et effectuez un paiement sur votre site, cela va réaliser un faux paiement.

<Warning>
  Pour avoir une installation fonctionnelle avec de vrais paiements, votre compte doit être activé par l'équipe Stan ! Nous faisons le nécessaire pour l'activer au plus vite. En attendant, vous pouvez désactiver l'extension "Stan Xpress Checkout" depuis l'interface de méthodes de paiement Woocommerce.
</Warning>
